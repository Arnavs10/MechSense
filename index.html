The Dataset

AI4I 2020 — 10,000
industrial sensor records

A real-world synthetic dataset reflecting genuine industrial machine behaviour, featuring five failure modes across three machine quality variants.

⚙️
Dataset Overview
10,000 data points spanning 3 machine types (H, M, L), 5 sensor features, and 5 distinct failure modes. Only 339 samples (3.39%) are failure cases — a classic class imbalance challenge that demands balanced weighting strategies.
339
Total Failures
Out of 10,000 observations — a 3.39% failure rate reflecting real industrial conditions.
🔬
5 Failure Modes
TWF (46), HDF (115), PWF (95), OSF (98), RNF (19) — each with distinct sensor signatures.
📡
6 Features Used
Machine Type, Air Temp, Process Temp, RPM, Torque, Tool Wear — all available in real-time from industrial sensors.
⚖️
28.5 : 1
Imbalance Ratio
Normal-to-failure ratio demands balanced class weighting — without it, models achieve 96.6% accuracy by predicting "no failure" every time.
